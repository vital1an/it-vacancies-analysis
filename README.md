# Анализ IT-вакансий

## Описание проекта
В рамках проекта выполнен анализ Data-вакансий с целью определения
востребованных навыков и требований к опыту работы на рынке труда.

Проект выполнен в учебных и портфельных целях и демонстрирует
процесс работы с реальными данными: от очистки до формирования выводов.

## Цель проекта
- Оценить распределение вакансий по географическому расположению
- Проанализировать требования работодателей к Data-специалистам  
- Выявить наиболее востребованные навыки  
- Оценить распределение вакансий по уровню опыта (Junior / Middle / Senior)

## Данные
Использован датасет с вакансиями в IT-сфере, содержащий:
- название вакансии  
- город  
- опыт работы  
- ключевые навыки
- размер заработной платы
- название валюты
- описание вакансии

### Источник данных
Данные получены с платформы **Kaggle**: https://www.kaggle.com/datasets/ilyazawilsiv/it-vacancies-from-headhunter-website/data

Из-за большого размера датасета и условий распространения
исходный файл **не размещён в репозитории**.

Для воспроизведения анализа необходимо:
1. Скачать датасет с Kaggle  
2. Поместить файл `IT_vacancies.csv` в папку `data`  
3. Запустить ноутбук `it_vacancies_analysis.ipynb`

## Этапы работы
1. Загрузка и первичный анализ данных
2. Категоризация названий Data специальностей  
3. Очистка и подготовка данных  
4. Категоризация опыта работы
5. Конвертация иностранных валют
6. Анализ распределения вакансий  
7. Анализ востребованных навыков  
8. Формирование выводов  

## Используемые технологии
- Python  
- Pandas  
- Matplotlib  
- Seaborn  
- Jupyter Notebook  

## Основные результаты
1. Чуть меньше половины вакансий приходятся на роль Data Analyst
2. Junior специалиста работодатели расматривают на удаленный формат работы меньше чем в 14% вакансий 
3. Data Engineer, Data Scientist и ML Engineer в среднем имеют дохов выше чем Data Analyst на 50% (На Middle и Senior грейдах)
4. Наиболее востребованные навыки среди всех Data-вакансий: Python, SQL, ML
5. Самые часто встречаемые пары навыков: Python+Sql, Sql+Анализ данных , ML+Python

## Ограничения анализа
1. Анализ выполнен на основе данных одного источника.
2. В части вакансий отсутствует информация о ключевых навыках.
3. Анализ уровня заработных плат проведён на основе ограниченного объёма данных, так как информация о зарплате указана не во всех вакансиях.
4. Данные охватывают короткий период времени, что не позволяет учитывать динамику рынка во времени.
